Sophia Commercial Door Lock Set - Unmatched Security for Durable Applications

Designed for both commercial and residential use, the Sophia Commercial Door Lock Set delivers top-tier security with its innovative non-handed design. This ensures that your doors remain secure without compromising ease of access or exit, making it a reliable solution for high-traffic areas and heavy-duty environments.

The SATIN CHROME finish not only enhances the aesthetic appeal but also offers a sleek, durable look that complements any space. Constructed from premium 304 stainless steel, this lock set is built to withstand the test of time, providing long-lasting performance even in challenging conditions.

With ANSI\/BHMA A156.13 Grade 1 certification, the Sophia lock set is trusted for its high security standards and over a million cycle durability. Its 3-hour fire rating makes it an essential addition for safety-conscious settings, ensuring your property is protected in emergencies.

Perfect for storerooms, offices, or other commercial spaces, this lock set features easy installation and access egress capabilities. It also includes re-keyable keys for added flexibility and ANSI Strike 1 compatibility for seamless integration with existing security systems.

Compliant with ADA accessibility standards, this product ensures user-friendly functionality for everyone. Whether you're securing your home or managing a commercial operation, the Sophia Commercial Door Lock Set is a smart, reliable choice for reliable security and long-term value.
