Berlin Modisch Zero Gravity Door Lever Handle Set

Elevate your door security and style with this sleek, modern door lever handle set from Berlin Modisch. Perfect for both residential and commercial settings, this set combines contemporary aesthetics with advanced functionality.

Key Features

  • Modern design that complements both interior and exterior doors
  • Hidden screws for a clean, sleek appearance
  • Reversible installation for left or right-handed doors
  • High-grade security with single cylinder deadbolt locking mechanism
  • Durable powder-coated iron black finish

Why Choose This Set?

This Zero Gravity door lever handle set is engineered for both beauty and security. Its hidden screws offer a refined, uninterrupted look while still providing reliable performance. The reversible design makes installation quick and simple, regardless of door orientation. Combining these elements with a high-security deadbolt, this set is ideal for homeowners and businesses seeking durable, modern door hardware that looks as good as it performs.

Modern Door Hardware with a Strong Focus on Security

The adjustable backset latch ensures compatibility with different door thicknesses, making it versatile for various applications. From front doors to office access points, this set delivers convenience and protection in one elegant package. The unique thumb-turn interior lock knob allows for easy access from the inside, while the single cylinder deadbolt provides secure locking from the outside with a KW1 keyway. Whether you're looking to enhance the visual appeal of your space or upgrade your security, this door lever handle set is a smart and stylish choice.
